# Responsible Release

This package releases benchmark inputs, code, parsed outcomes, aggregate results, and validation summaries.

The open GitHub package intentionally excludes raw generated attacker/defender/gate dialogue and raw provider response folders. Those artifacts can contain fabricated statistics, unsupported citations, false attributions, or risk-sensitive manipulative framing. They are not needed to reproduce the main denominator-explicit reporting tables.

If raw dialogue is shared later, use a screened-access process:

1. Share only for bona fide research or audit purposes.
2. Require users to agree not to use the material for persuasion targeting, fraud, harassment, or evasion.
3. Keep fabricated-statistic, unsupported-citation, and false-attribution examples contextualized with labels and warnings.
4. Avoid publishing full attacker-defender pairing matrices together with raw intervention text.

Public aggregates and parsed outcomes are suitable for GitHub release.
